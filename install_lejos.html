Lego Mars Rover
LeJOS is the system that we use on the lego EV3 robots, to use our program your EV3 will need LeJOS.
Pre-requisites
1. EV3 Robot
2. One MicroSD card between 2GB and 32GB formatted in FAT32
3. JAVA SDK (Not JRE) Download Here
Installing on windows
1. Download the LeJOS installer from here
2. Click next
3. Select the JDK Directory
4. Click next
5. Click Install
6. Make sure that the "Launch EV3SDCard utility" is ticked 7. Click Finish 8.
Windows Install, Wizard, sdCard utility
Mac/Linux, unpack tar.gz, write sd500.img using disk image writing utility
